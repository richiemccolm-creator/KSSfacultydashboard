How to use Tracking & Monitoring

Quick start

  1. Data loads automatically when you open this page. Use Refresh if scores look out of date.
  2. Start on Overview — use Faculty insights after refresh for per-subject gaps, then drill into Classes or At Risk.
  3. Go to Classes and click a teacher to see their classes and pupils.
  4. Click any pupil to open their profile drawer with charts and notes.

Key terms

TP (tracking point)
A scoring period such as TP1 or TP2 in the tracker.
At risk
A pupil with 2 or more dimensions averaging 2.5 or below across all tracking points.
Dimensions
Creating, Presenting, Evaluating, Effort, Behaviour, Home learning (Art may use Progress).
Coverage
Green squares = most pupils scored for that TP; amber = partial; grey = missing.
Score scale
1–4 where higher is stronger. Concern ≤2.5, Secure 2.5–3.25, Strong >3.25.

Exports

  • Faculty insights → Export Excel — summary tables plus pupil-level averages (opens in Excel).
  • Faculty insights → Print summary — printable attainment and gap report per subject.
  • Export (header) — all filtered score rows for analysis.
  • Pupil List → Export CSV — one row per pupil with TP columns.
  • More views → Print at-risk digest — printable at-risk list.
