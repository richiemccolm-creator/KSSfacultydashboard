# Senior Phase Tracker (archived prototype)

This standalone HTML prototype was removed from Faculty Hub on 2026-06-03. It used mock pupil/checkpoint data and was not connected to cloud storage.

**To restore in the hub:** copy `senior-phase-tracker.html` back to the repo root and re-wire `embed-senior-phase-tracker` in `faculty-hub.html`, `dip-tracker-app.js`, and `hgios4-toolkit-app.js` / `hgios4-toolkit-hub-bridge.js`.

**Open locally:** open `senior-phase-tracker.html` in a browser (no server required).
